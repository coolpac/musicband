# Приветственное видео для Telegram User Bot

Сюда помещается оптимизированное видео, которое бот отправляет при команде `/start` (приветственное сообщение).

## Как добавить видео

1. Положите исходный файл (например, `2026-02-12 21.59.07.mp4`) в удобное место.
2. Запустите скрипт оптимизации из корня проекта:

   ```bash
   ./scripts/optimize-welcome-video.sh "путь/к/вашему/видео.mp4"
   ```

   Скрипт создаст файл `welcome.mp4` в этой папке, оптимизированный для Telegram (лимит 50 МБ, 720p, быстрый старт воспроизведения).

3. Если файл `welcome.mp4` отсутствует, бот при `/start` отправит только текстовое приветствие без видео.

## Требования

- **ffmpeg** должен быть установлен в системе (`brew install ffmpeg` на macOS).
